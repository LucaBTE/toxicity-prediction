How to get started, use and be able to modify the report directly inside Visual Studio Code using LaTex.

For windows I folloed this [tutorial](https://www.youtube.com/watch?v=4lyHIQl4VM8&t=417s): 
1. Install [MikText](https://miktex.org/download)
2. Install [Pearl](https://strawberryperl.com/)
3. Install the estension ["LaTex Workhop"](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop)
4. Reboot the PC
5. Open in VSCode the folder with the .tex file
